# üçé Fruit Classification AI - Computer Vision Project

A Deep Learning-powered web application that classifies images of fruits into 7 distinct categories. Built with **Django**, **TensorFlow/Keras**, and **MobileNetV2** (Transfer Learning) for high accuracy.

**Repository:** [https://github.com/MuhammadAsad878/fruit-classification-model-cv.git](https://github.com/MuhammadAsad878/fruit-classification-model-cv.git)

---

## üöÄ Features
* **7-Class Classification:** Detects Apple, Banana, Mango, Orange, Pineapple, Strawberry, and Watermelon.
* **High Accuracy:** Uses **MobileNetV2** (Transfer Learning) trained on a custom dataset.
* **Web Interface:** Clean, responsive UI built with **Django** and **Bootstrap 5**.
* **Confidence Score:** Displays the model's certainty percentage.
* **Voice Feedback:** The application announces the detected fruit using text-to-speech.
